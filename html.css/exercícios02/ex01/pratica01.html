Tipos de Formatações CSS

  • Método Externo
  • Método Interno
  • Método Inline
Método Externo
Este método é o mais usado, porque ele nos permite mudificar HTML com muita facilidade.
Por estar em outra página, o primeiro método mais usado, mais ligado com linkadas.

Método Interno
O método Interno permanece na mesma página,
é o segundo método mais usado.

Método Inline
Método Inline é o método especificamente para um documento com uma quantidade mínima de páginas,
por ser uma configuração de Tags em Tags.
